⚔ WAVE PLANNER

A live war planning tool for Utopia kingdoms.
Runs inside the Intel Site — no separate login needed.
1
Enter your GitHub username and repo name (where you uploaded app.js and this index.html):
2
Drag this button to your bookmarks bar:
⚔ Wave Planner
Drag to bookmarks bar, or right-click → "Bookmark this link"
3
Open intel.utopia.site via your in-game link (so you're logged in)
4
Click the ⚔ Wave Planner bookmark — the tool opens as an overlay
5
War leader: WAR BOARD tab → drag enemy provinces into waves → assign ops → Save Plan

Players: MY ORDERS tab → see targets and attack matchups for your attacker
How sharing works:
War leader saves the plan → stored on Kagge's Intel Site backend.
All kingdom members click the same bookmark → everyone sees the same wave assignments.
Hit ↻ Refresh inside the tool to pull fresh intel at any time.

Share this page URL with your whole kingdom so they can all install the bookmark.
